Вопросы и Ответы

Вопрос 1: Что такое workflow?Для чего он нужен? Как работает ?
Ответ: это команды которые выполняются совместно членами команды.Он нужен для того чтобы все выполнялось по плану и чтобы было легче отслеживать за процессом

Вопрос 2: Что такое git rebase?Для чего он нужен?Разница между git rebase и git merge?
Ответ: git rebase помогает изменять историю коммитов с помощью перемещения!В отличие от merge Rebase переписывает историю коммитов

Вопрос 3: Что такое git fetch?Разница между git fetch и git pull?
Ответ: Он помогает загрузить коммиты,файлы из удаленного репозитория в локальный.Git pull загружает удаленное содержимое и сразу изменяет состояние локального репозитория
